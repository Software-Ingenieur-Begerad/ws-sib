Softwarefreiheit

Wenn {{ site.organization.short }} von Softwarefreiheit spricht, dann ist die Rede von Freiheit im Sinne von Unabhängigkeit und NICHT Kostenfreiheit.

{{ site.organization.short }} verwendet den Begriff Softwarefreiheit, um den selbstbestimmten Umgang mit Software zu beschreibt. Weitere Synonyme sind Freie Software, Open Source, FOSS, Libre Software oder FLOSS. Details zu diesen Synonymen können hier nachgelesen werden.

Aus Softwarefreiheit resultiert ein Alleinstellungsmerkmal von {{ site.organization.short }}, welches Kunden den selbstbestimmten Umgang mit Investitionen bzw. Ressourcen ermöglicht. Beispiele hierfür können im Blog nachgelesen werden.

Ermöglicht wird diese Selbstbestimmung durch die vier grundlegenden Freiheiten, welche Kunden von freier Software garantiert werden:

  • Verwenden,
  • Verstehen,
  • Verbreiten und
  • Verbessern.
{{ site.organization.short }} und die Free Software Foundation Europe (FSFE) teilen dieses Verständnis. Details hierzu können auf den FSFE Webseiten nachgelesen werden.
